# 已确认规则摘要

这份文档用于实现时快速查阅。完整设计说明见 [CoreGameplay.md](CoreGameplay.md)。

## 基础定位

- 游戏是 1-4 人合作的 2D 俯视角本地多人游戏。
- 玩家在飞船内部移动，主要使用手柄操作。
- 当前版本只验证核心合作玩法。
- 暂不加入间谍、隐藏目标、手柄震动、结算指认等扩展规则。

## 胜负条件

- 胜利：飞船在护盾归零前抵达终点。
- 失败：飞船护盾归零。

## 飞船

- 飞船根据船体当前旋转方向前进。
- 燃料充足时正常前进。
- 燃料较少时速度降低。
- 燃料耗尽时停止前进。
- 船体撞到墙面、障碍物或炸药桶会扣除护盾。
- 护盾值就是飞船生命值。

## 玩家

- 支持 1-4 名玩家。
- 玩家不绑定固定岗位，任何玩家都能操作任何装置。
- 船内墙壁影响移动路线。
- 船内墙壁不阻止玩家携带资源。
- 玩家之间允许互相碰撞。
- 玩家碰撞应制造轻微推挤和拥挤感，避免严重卡死。

## 玩家携带资源

- 玩家可以携带燃料。
- 玩家可以携带护盾。
- 燃料携带上限为 3。
- 护盾携带上限为 3。
- 抓到燃料或护盾后，资源进入操作玩家身上。
- 资源必须投入对应装置后才会补充飞船。
- 如果对应资源已满，第一版建议不增加资源。

## 锚点

- 飞船有四个局部方向锚点：上、下、左、右。
- 四个方向跟随船体旋转，不固定对应屏幕方向。
- 每个锚点都有对应船内操作装置。
- 玩家必须站到装置上才能操作对应锚点。
- 操作锚点时，玩家不能同时做其他交互。
- 锚点可用于抓墙转向，也可用于抓漂浮物。

## 锚操作

- 锚点操作参考《黄金矿工》爪子逻辑。
- 锚围绕自身局部基础方向旋转偏移。
- 第一版锚点可偏移角度暂定 90 度。
- 第一版锚最大发射距离暂定 15 单位。
- 锚碰到任何物品都会立即结算并收回。
- 锚未命中目标，到达最大距离后收回。
- 锚抓到东西后不会持续停在外部拉扯。

## 抓墙

- 锚抓到墙面或可抓地形时，飞船获得一次转向冲量。
- 转向冲量会改变飞船航向。
- 转向冲量也会改变船体朝向。
- 抓墙效果是一次性修正，不是持续拖拽。

## 漂浮物

- 当前漂浮物包括：燃料、护盾、垃圾、蛛网、炸药桶。
- 燃料被锚抓到后，进入操作玩家身上。
- 护盾被锚抓到后，进入操作玩家身上。
- 垃圾被锚抓到后直接消失，只浪费一次抓取机会。
- 蛛网被锚抓到后，只减慢命中它的那个锚点。
- 炸药桶被锚碰到时，在外部原地爆炸，飞船扣护盾。
- 炸药桶不会被拉回船上。

## 第一版简化建议

- 碰撞伤害先统一处理为固定扣护盾。
- 资源满时先不增加资源，不做掉落和溢出逻辑。
- Restart、结算、热插拔恢复等流程后续再细化。
- 先把船内跑动、锚点操作、抓墙转向、燃料护盾压力做出来。
